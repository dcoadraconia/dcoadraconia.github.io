Unofficial Draconia Advent Event

What is the Draconia Advent Event?
- The "Draconia Advent Event" is an unofficial collection of Draconia: Curse of Anti-Magic fan content created by Big-T Rex and uploaded to his Draconia fan website (where you are right now.) It is mainly composed of Draconia fanfiction and some teasers for Draconia fandom stuff Rex is still working on.


What the heck is a Draconia?
- If you don't know what Draconia is, umm, you're in the wrong place.
Draconia: Curse of Anti-Magic is a personal story project created by one person, and she's shared her stories about it with a handful of people (Big-T Rex being one of them). If you haven't heard about Draconia, you won't understand anything happening here. Sorry.


How do I view the Event content?
- The Event's content is spread across a number of different "secret pages" on this website. To access a "secret page," enter into your web browser the web address of this website (https://dcoadraconia.github.io) followed by a "/" and then the name of the page. For example, you could try to access the secret page "secret" by going to https://dcoadraconia.github.io/secret. If the page exists, then it will load and display to your device. Otherwise (such as in this example page, which isn't a real secret page) you will be taken to the GitHub Pages default 404 page, meaning that the page you are trying to view does not exist.


Things to remember when finding secret pages
- Each secret page is at an address of the form
"https://dcoadraconia.github.io/<secret>"
where "<secret>" is replaced with the name of the page. Secret page names are one word, so no spaces (" ") or underscores ("_").
Furthermore, all secret page names must be typed in fully lowercase and no uppercase characters, or else GitHub Pages will not recognize the name of the page. (Use "secret", not "Secret" or "SECRET".)
